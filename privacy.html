Privacy Policy – Trafikverket Exam Slot Notifier

Last updated: July 31, 2025

This Chrome extension helps users monitor available driving exam time slots on Trafikverket’s website. It does not collect, transmit, or store any personal data beyond what is necessary to trigger local notifications.

1. Information Collected

  • No personal data is collected.
  • User preferences (e.g., refresh interval) are handled locally in the browser cache only.
  • No data is stored permanently via chrome.storage or synced across devices.

2. Data Usage

No data is transmitted to external servers. If the user opts in, the extension can send notification alerts via email, based only on data the user explicitly provides (e.g., their email address). This email service is user-initiated; no background email collection occurs.

3. Permissions Used

  • scripting / content_scripts — to interact with the Trafikverket website and detect available test slots.
  • identity — not used to collect personal data; only employed if users sign in (optional). You may remove it if not implemented.
  • notifications — to provide desktop alerts when exam slots become available.
  • storage — reserved but not actively used to store user preferences since nothing is persisted via chrome.storage.

4. Notifications

Notifications are shown only when a new exam time slot is detected. Alerts are triggered by user‑initiated checks and display minimal details (e.g. date/time of slot). Notifications appear only while the browser is active; no background tracking is performed.

5. Data Sharing & Third Parties

No data is shared with any third parties. No analytics, tracking, or advertising tools are used. If email notifications are enabled, they are sent directly by the client and not stored or shared with any external service by default.

6. Security

All processing is done locally within your Chrome browser. No data is transmitted or stored externally unless the user explicitly uses the email notification feature. The extension does not collect browsing history or track user activity outside Trafikverket.

7. GDPR & CCPA Compliance

Since no personal or sensitive data is collected or processed, this extension does not fall under GDPR or CCPA data collection requirements. Users retain complete control—there is nothing to request or delete, as no persistent data exists.

8. User Rights

As no personal data is processed, rights to access, amendment, or deletion of personal data do not apply. Users can manage notification behavior via Chrome’s notification settings or through extension options.
